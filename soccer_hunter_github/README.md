# Soccer Hunter - منصة تسويق لاعبي كرة القدم والمدربين

منصة متخصصة لتسويق لاعبي ومدربي كرة القدم، توفر واجهة سهلة الاستخدام لعرض المواهب والتواصل بينهم وبين الأندية والوكلاء.

## المميزات

- إدارة ملفات اللاعبين والمدربين
- دعم متعدد اللغات (العربية، الإنجليزية، وغيرها)
- مؤتمرات فيديو مباشرة
- نظام محادثات
- لوحة تحكم مشرفين
- تحميل ومشاركة الصور والفيديوهات والمستندات
- جدولة المواعيد والاجتماعات

## متطلبات النظام

- PHP 7.4 أو أعلى
- MySQL 5.7 أو أعلى
- خادم ويب Apache أو Nginx

## التثبيت

1. قم بتنزيل أو استنساخ هذا المستودع
2. قم بتحميل الملفات إلى خادم الويب
3. قم بزيارة http://your-domain.com/setup/install.php لبدء عملية التثبيت
4. اتبع التعليمات لإعداد قاعدة البيانات وحساب المشرف

للمزيد من المعلومات، راجع [دليل التثبيت](docs/INSTALLATION.md).
