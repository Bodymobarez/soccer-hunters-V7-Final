# Soccer Hunter - منصة تسويق اللاعبين والمدربين

## نظرة عامة
Soccer Hunter هو منصة متخصصة لتسويق لاعبي ومدربي كرة القدم، توفر واجهة سهلة الاستخدام لعرض المواهب والتواصل بينهم وبين الأندية والوكلاء.

## متطلبات النظام
- PHP 7.4 أو أعلى
- MySQL 5.7 أو أعلى
- خادم ويب Apache أو Nginx
- تمكين خدمة GD للصور
- تمكين امتدادات PDO لقواعد البيانات

## خطوات التثبيت
1. قم بتحميل الملفات إلى مجلد على خادم الويب الخاص بك
2. أنشئ قاعدة بيانات جديدة في MySQL
3. قم بتعديل ملف التكوين في  بمعلومات الاتصال بقاعدة البيانات
4. قم بتشغيل سكريبت إنشاء قاعدة البيانات من خلال استيراد ملف 
5. قم بضبط الصلاحيات للمجلدات التالية لتكون قابلة للكتابة:
   - uploads/images/
   - uploads/videos/
   - uploads/documents/

## معلومات تسجيل الدخول الافتراضية
- اسم المستخدم: admin
- كلمة المرور: admin123

## ميزات النظام
- تسجيل وتسجيل دخول المستخدمين مع أدوار مختلفة
- عرض المواهب حسب الفئات (لاعبين، مدربين، حراس مرمى، الخ)
- البحث في قاعدة بيانات المواهب
- رفع وإدارة الوسائط المتعددة (صور، فيديوهات، مستندات)
- جدولة المواعيد والاجتماعات
- دعم مؤتمرات الفيديو المباشرة
- نظام مراسلة داخلي
- دعم متعدد اللغات
- لوحة تحكم للمشرفين
