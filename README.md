# Soccer Hunter

منصة متكاملة لإدارة المواهب الرياضية ولاعبي كرة القدم، تعمل بنظام متعدد اللغات.

## المميزات

- دعم متعدد اللغات (العربية، الإنجليزية، الفرنسية، الألمانية، البرتغالية، الإسبانية، الإيطالية، اللاتينية)
- نظام إدارة لاعبين ومدربين متكامل
- منصة لوكلاء اللاعبين والأطباء الرياضيين
- تسجيل للأندية للتعاقد مع اللاعبين والمدربين
- نظام محادثات فورية
- نظام مؤتمرات فيديو مدمج مع إمكانية التسجيل
- رفع ملفات (صور، فيديوهات، سير ذاتية) للاعبين والمدربين
- معلومات عن الملاعب وأخبار كرة القدم

## التقنيات المستخدمة

- واجهة أمامية: React
- واجهة خلفية: Node.js/Express
- قاعدة بيانات: PostgreSQL مع Drizzle ORM
- نظام المصادقة: Passport.js
- التصميم: Tailwind CSS مع shadcn/ui
- دعم اللغات: i18n مع منظومة ترجمة مخصصة
- مؤتمرات الفيديو: WebRTC
- المحادثات الفورية: WebSockets

## متطلبات التشغيل

- Node.js v18 أو أحدث
- PostgreSQL

## تثبيت وتشغيل المشروع

```bash
# تثبيت التبعيات
npm install

# إنشاء قاعدة البيانات
npm run db:push

# تشغيل في بيئة التطوير
npm run dev

# بناء للإنتاج
npm run build

# تشغيل في بيئة الإنتاج
npm start
```

## الإصدار PHP

يتوفر أيضًا إصدار PHP من المشروع في مجلد `php_version`، مع قاعدة بيانات MySQL.
